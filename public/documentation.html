Rate Flow

The Currency Conversion API is easy to use and requires no API key, making it very easy to integrate.

This API was developed to simplify the process of fetching exchange rate data for developers. Whether you're building a currency converter, a financial dashboard, or an e-commerce platform, this API provides the data needed to handle currency conversions efficiently.

This API uses the Open Exchange Rate API.

I built this API to offer a solution for developers like myself to access exchange rate data easily, as a learning tool, for trying out new programming languages or technology, or easy prototyping.

/codes.json

Description: Get a list of all supported currency codes. 

GET /codes.json

Example Response:

{
    "USD": "United States Dollar",
    "EUR": "Euro",
    "GBP": "British Pound Sterling",
    "JPY": "Japanese Yen"
}

/latest.json

Description: Fetch the latest exchange rates for one or more base currencies. You can specify symbols to get rates only for certain currencies. 

GET /latest.json?base=USD,EUR&symbols=GBP,JPY

Query Parameters:

  • base: A comma-separated list of base currencies (e.g., "USD,EUR"). Defaults to "USD" if not provided.
  • symbols: A comma-separated list of target currencies (e.g., "GBP,JPY") or JSON object for multiple bases (e.g., {"USD": ["GBP"], "EUR": ["JPY"]}).

Example Response:

[
    {
        "date": "2023-10-01",
        "base": "USD",
        "rates": {
            "GBP": 0.74,
            "JPY": 110.47
        }
    },
    {
        "date": "2023-10-01",
        "base": "EUR",
        "rates": {
            "JPY": 130.54
        }
    }
]

/historical/{date}.json

Description: Fetch historical exchange rates for one or more base currencies on a given date. 

GET /historical/2023-09-30.json?base=USD,EUR&symbols=GBP,JPY

Path Parameter:

  • date: The date for historical rates in "YYYY-MM-DD" format.

Query Parameters:

  • base: A comma-separated list of base currencies (e.g., "USD,EUR"). Defaults to "USD" if not provided.
  • symbols: A comma-separated list of target currencies (e.g., "GBP,JPY") or JSON object for multiple bases (e.g., {"USD": ["GBP"], "EUR": ["JPY"]}).

Example Response:

[
    {
        "date": "2023-09-30",
        "base": "USD",
        "rates": {
            "GBP": 0.74,
            "JPY": 110.47
        }
    },
    {
        "date": "2023-09-30",
        "base": "EUR",
        "rates": {
            "JPY": 130.54
        }
    }
]

/convert.json

Description: Convert an amount from one currency to another. 

GET /convert.json?from=USD&to=EUR&amount=100

Query Parameters:

  • from: The base currency code (e.g., "USD").
  • to: The target currency code (e.g., "EUR").
  • amount: The amount to convert.

Example Response:

{
    "from": "USD",
    "to": "EUR",
    "amount": 100,
    "convertedAmount": 85
}
