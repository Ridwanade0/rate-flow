History Rates API

This API retrieves historical currency rates based on a specified date and currency symbols.

Endpoint

GET /api/history-rates/:date

Request Parameters

  • date (path parameter): string - The date for which historical rates are requested. Format: YYYY-MM-DD

Query Parameters

  • base (query parameter): string - A comma-separated list of base currencies (default: USD).
  • symbols (query parameter): string or JSON - A comma-separated list or JSON object specifying the target currency symbols to retrieve. If omitted, all rates for the base currency will be returned.

Response Format


{
    "code": 200,
    "success": true,
    "data": [
        {
            "date": "YYYY-MM-DD",
            "base": "CURRENCY_CODE",
            "rates": {
                "TARGET_CURRENCY": RATE,
                ...
            }
        },
        ...
    ]
}

Response Example


{
    "code": 200,
    "success": true,
    "data": [
        {
            "date": "2023-10-01",
            "base": "USD",
            "rates": {
                "EUR": 0.85,
                "JPY": 110.50
            }
        },
        {
            "date": "2023-10-01",
            "base": "GBP",
            "rates": {
                "USD": 1.35,
                "JPY": 149.00
            }
        }
    ]
}

Error Responses


{
    "code": 400,
    "success": false,
    "message": "Invalid date format. Please use YYYY-MM-DD."
}
        

{
    "code": 500,
    "success": false,
    "message": "Error message detailing the issue."
}

Usage Example


GET /api/history-rates/2023-10-01?base=USD,GBP&symbols={"USD": ["EUR", "JPY"], "GBP": ["USD"]}

Notes

  • Ensure the date is in the correct format.
  • Multiple base currencies can be specified.
  • The API will return rates only for the requested symbols, if specified; otherwise, all rates will be returned for the base currencies.
