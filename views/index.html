Rate Flow

API Introduction

Rates Flow provides a simple, lightweight, and portable JSON API that supplies live and historical foreign exchange (forex) rates. This API is easy to integrate and presents data in JSON format. Rates are fetched and delivered from Open Exchange Rates, which compiles and blends rates algorithmically from various trusted sources to ensure consistent and unbiased accuracy.

The exchange rates provided by the Rates Flow API are sourced from Open Exchange Rate, blended together, and delivered in JSON format for everyone to access. There are no complicated queries, confusing authentication processes, or long-term contracts involved.

End-of-day rates are available historically from October 21, 2024, onwards. However, some dates may be unavailable because this project is open-source and hosted on a free-tier Docker container, which shuts down when not in use, resulting in gaps in the data.

In addition to being in development and intended for learning purposes, this project is open-source, allowing you to contribute or self-host it if you wish.
